## 낭눈 봇 기능 목록

### 1. 기본 명령어
- `!안녕` -> "안녕하세요!"
- 추가 예정

### 2. 포춘 쿠키
- `!포춘쿠키`
- 랜덤한 포춘쿠키 문장 출력

### 3. 녜힁제조기
- `!녜힁`
- [녜힁제조기](https://beepman.github.io/nyehuing/)에서 생성한 녜힁 출력

### 4. 로또게임
#### 4.1. `!출석체크`
- 하루에 한 번 포인트 지급
#### 4.2. `!로또구매 [개수]`
- 입력한 개수만큼 랜덤한 로또 발행
#### 4.3. `!로또결과`
- 당첨 번호와 비교하여 로또 결과 출력
- 당첨 결과에 맞는 포인트 지급
#### 4.4. `!랭킹`
- 서버 내의 사용자들을 포인트가 많은 순으로 랭킹 출력
